# Compteur calorique & recettes

PWA de tracking de macros et recettes pour la sèche.

**Accès :** https://ledoreamon.github.io/compteur-calorique-recettes/

## Features
- 📊 Tracker de macros journalier (J−7 à J+1)
- 🍳 Recettes avec calcul automatique des ingrédients
- 📷 Analyse photo d'assiette par IA
- ✦ Estimation macros desserts / repas libres par IA
- 💾 Sauvegarde automatique (localStorage)
- 📱 Installable comme app (PWA)

## Setup clé API
Au premier lancement, tape sur ⚙ pour configurer ta clé API Anthropic (nécessaire pour les features IA).
